Basic Excel (ต่อ)

🧮 Logic & Conditional Aggregation

ตรรกะและการสรุปผลแบบ "มีเงื่อนไข" — หัวใจของการวิเคราะห์ข้อมูลใน Excel

1) IF / IFS — ตรวจเงื่อนไขทีละค่า

ลากเพื่อกำหนดผลผลิต (กก.) — IF ตรวจ 2 ทางเลือก ส่วน IFS ตรวจหลายเกณฑ์เรียงลำดับ (≥ 700 = A, ≥ 500 = B, ต่ำกว่า = C)

Text Functions

✂️ จัดการและทำความสะอาดข้อความ

ในฟาร์มมักมี "รหัสล็อตสินค้า" ที่ต้องแยกส่วน ลองพิมพ์รหัสแล้วดูว่าแต่ละฟังก์ชันตัด/แปลงอย่างไร

ตัวอย่างมีเว้นวรรคหน้า-หลังไว้ให้เห็นผลของ TRIM · โครงสร้าง = พืช(3)-แปลง-ปี · สังเกตแถวแรก ๆ: ถ้าไม่ TRIM ก่อน LEFT จะติดเว้นวรรคมาด้วย!

Date & Time Functions

📅 ทำงานกับวันที่และเวลา

สำคัญมากในงานโลจิสติกส์/ฟาร์ม เช่น นับวันทำงานจนถึงวันเก็บเกี่ยว หรือหาว่าวันส่งของตรงกับวันอะไร

💡 WEEKDAY แบบมาตรฐาน: 1 = อาทิตย์ … 7 = เสาร์ · NETWORKDAYS นับเฉพาะวันจันทร์–ศุกร์ (ไม่รวมเสาร์-อาทิตย์)

Lookup Functions

🔎 ค้นหาและดึงข้อมูลข้ามตาราง

เลือกรหัสแปลง แล้วให้ Excel "ค้น" ข้อมูลของแปลงนั้นจากตาราง Harvest Log มาแสดง

VLOOKUP ค้นจากคอลัมน์ซ้ายสุด · XLOOKUP ยืดหยุ่นกว่า ค้นได้ทุกทิศและไม่ต้องนับเลขคอลัมน์ · INDEX+MATCH = MATCH หา "แถวที่เท่าไร" แล้ว INDEX ดึงค่าจากแถวนั้น — ใช้ได้ทุกเวอร์ชัน

Data Toolkit

🧰 เครื่องมือจัดการข้อมูล — ไม่ต้องเขียนสูตร

เครื่องมือสำเร็จรูปในแท็บ Data และ Home ของ Excel — ใช้ทำความสะอาดและควบคุมคุณภาพข้อมูลก่อนวิเคราะห์ (ต่อจากเรื่องธรรมาภิบาลข้อมูลครั้งที่ 1)

1) Text to Columns — แยก 1 คอลัมน์เป็นหลายคอลัมน์

รหัสล็อต "พืช-แปลง-ปี" ถูกกรอกรวมในคอลัมน์เดียว ลองกดปุ่มเพื่อแยกด้วยตัวคั่น "-"

2) Remove Duplicates — ลบแถวที่ซ้ำกัน

ทะเบียนแปลงถูกกรอกซ้ำ (แถวสีแดง) — เครื่องมือนี้เก็บแถวแรกไว้ ลบที่เหลือ

3) Flash Fill — เติมตามแบบอย่าง (Ctrl+E)

พิมพ์ "ผลลัพธ์ตัวอย่าง" ให้ Excel ดู 1 แถว (ดึงรหัสแปลงจากรหัสล็อต) แล้วกด Ctrl+E — Excel เดารูปแบบและเติมที่เหลือให้

4) Data Validation — คุมค่าที่กรอกได้

ช่อง "เกรด" ตั้ง Data → Data Validation → List = A, B, C — ลองพิมพ์ค่าอื่นดู

5) Conditional Formatting — heatmap ความชื้นดิน (%)

Home → Conditional Formatting → Color Scales · สีแดง = ดินแห้ง ต้องรีบให้น้ำ · สีเขียว = ความชื้นเหมาะสม — เห็นปัญหาได้โดยไม่ต้องอ่านตัวเลขทีละช่อง

Advanced Excel

📊 PivotTable — สรุปข้อมูลในไม่กี่คลิก

เครื่องมือสรุปข้อมูลที่ทรงพลังที่สุดของ Excel · สรุป ผลผลิต / ยอดขาย / อัตราสูญเสีย ราย เดือน / ชนิดพืช / แปลง — เลือกฟิลด์ลงแต่ละโซน แล้วดูตารางสรุปและกราฟเกิดขึ้นทันที (จำลองการลาก-วางฟิลด์)

💡 นี่คือแนวคิดเดียวกับการลากฟิลด์ลงโซน Rows / Values / Filters ในหน้าต่าง PivotTable จริง · เพิ่มกราฟได้ทันทีด้วย PivotChart
