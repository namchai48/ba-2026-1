Basic Excel (ต่อ)

🧮 Logic & Conditional Aggregation

ตรรกะและการสรุปผลแบบ "มีเงื่อนไข" — หัวใจของการวิเคราะห์ข้อมูลใน Excel

1) IF — ตรวจเงื่อนไขทีละค่า

ลากเพื่อกำหนดผลผลิต (กก.) แล้วดูว่าเมล่อนลูกนี้จัดเป็นเกรดไหน (เกณฑ์ ≥ 700 = พรีเมียม)

→

2) SUMIFS / COUNTIFS — สรุปเฉพาะแถวที่ตรงเงื่อนไข

–
COUNTIFS (จำนวนแถว)
–
SUMIFS (ผลรวม กก.)
–
AVERAGEIFS (เฉลี่ย)
Text Functions

✂️ จัดการและทำความสะอาดข้อความ

ในฟาร์มมักมี "รหัสล็อตสินค้า" ที่ต้องแยกส่วน ลองพิมพ์รหัสแล้วดูว่าแต่ละฟังก์ชันตัด/แปลงอย่างไร

ตัวอย่างมีเว้นวรรคหน้า-หลังไว้ให้เห็นผลของ TRIM · โครงสร้าง = พืช(3)-แปลง-ปี

Date & Time Functions

📅 ทำงานกับวันที่และเวลา

สำคัญมากในงานโลจิสติกส์/ฟาร์ม เช่น นับวันทำงานจนถึงวันเก็บเกี่ยว หรือหาว่าวันส่งของตรงกับวันอะไร

💡 WEEKDAY แบบมาตรฐาน: 1 = อาทิตย์ … 7 = เสาร์ · NETWORKDAYS นับเฉพาะวันจันทร์–ศุกร์ (ไม่รวมเสาร์-อาทิตย์)

Lookup Functions

🔎 ค้นหาและดึงข้อมูลข้ามตาราง

เลือกรหัสแปลง แล้วให้ Excel "ค้น" ข้อมูลของแปลงนั้นจากตาราง Harvest Log มาแสดง

VLOOKUP ค้นจากคอลัมน์ซ้ายสุด · XLOOKUP ยืดหยุ่นกว่า ค้นได้ทุกทิศและไม่ต้องนับเลขคอลัมน์
เลือกใช้ตัวไหน?
XLOOKUP
Excel รุ่นใหม่ — ใช้ตัวนี้ก่อน เขียนสั้นและยืดหยุ่นที่สุด
INDEX + MATCH
ทางเลือกที่ทำงานได้ทุกเวอร์ชัน แทน VLOOKUP ได้
Dynamic Arrays

⚡ สูตรที่คืนผลลัพธ์หลายค่าในครั้งเดียว

พิมพ์สูตรช่องเดียว ผลลัพธ์จะ "หก" (spill) ออกมาหลายแถวและอัปเดตเองเมื่อข้อมูลเปลี่ยน

Advanced Excel

📊 PivotTable — สรุปข้อมูลในไม่กี่คลิก

เครื่องมือสรุปข้อมูลที่ทรงพลังที่สุดของ Excel · เลือกฟิลด์ลงแต่ละโซน แล้วดูตารางสรุปและกราฟเกิดขึ้นทันที (จำลองการลาก-วางฟิลด์)

💡 นี่คือแนวคิดเดียวกับการลากฟิลด์ลงโซน Rows / Values / Filters ในหน้าต่าง PivotTable จริง · เพิ่มกราฟได้ทันทีด้วย PivotChart
